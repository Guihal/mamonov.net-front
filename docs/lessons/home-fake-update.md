# Урок: Фейковое обновление

**Slug**: `home-fake-update`
**Категория**: Дом (home)
**Порядок**: 2 из 3
**Угроза**: Вредоносное ПО через фейковые обновления

---

## Идея

Пользователь листает новостной сайт в браузере. Поверх страницы всплывает агрессивный попап:
**«⚠️ ВНИМАНИЕ! Ваш Flash Player устарел! Компьютер в опасности! Скачайте обновление прямо сейчас!»**

Нужно: не паниковать, не нажимать кнопки в попапе, проверить откуда он взялся, закрыть его.

---

## Программы

`browser`

## Сценарий по шагам

### Шаг 0 — старт

- Браузер открыт на `https://news-daily.ru` (мок-новостной сайт `FakeNewsPage.vue`)
- Через ~1.5с поверх страницы появляется оверлей-попап со стилем «скама» (красный фон, мигающий значок ⚠️):
  - Заголовок: «Ваш Flash Player версии 11.2 УСТАРЕЛ!»
  - Подзаголовок: «Обнаружена угроза безопасности. Немедленно скачайте обновление.»
  - Кнопка (большая, зелёная): «Скачать обновление (3.7 МБ)»
  - Кнопка (маленькая, серая): «Напомнить позже»
  - Внизу мелким шрифтом: `update-flash-now.site/download`
- Адресная строка браузера при этом показывает: `https://news-daily.ru` (пользователь сам не переходил никуда)

**Маскот — очередь реплик при старте:**

1. `{ text: 'Ты читаешь новости...', emotion: 'neutral', delay: 1500 }` — авто
2. `{ text: 'Хм, что-то всплыло! Не торопись нажимать — сначала разберись, что это такое.', emotion: 'worried' }` — ждёт клика

---

### Шаг 1 — пользователь изучает попап

Доступны Quick Actions:

- **«Проверить URL»** → открывает `tools://url-checker` в новой вкладке → шаг 2
- **«Закрыть предупреждение»** → сразу закрывает попап → переход к шагу 3 (тоже правильно, но без бонусной проверки)

Маскот при наведении/зависании над кнопкой «Скачать» (если реализуемо — `onPopupHover` ивент):

- `{ text: 'Стоп! Ты серьёзно собираешься нажать туда?', emotion: 'worried', delay: 3000 }` — авто

---

### Шаг 2 — проверка URL (URL Checker)

Пользователь вводит `update-flash-now.site` в инструмент проверки URL.

Результат (мок): **Опасно** — домен зарегистрирован 3 дня назад, не является официальным сайтом Adobe, входит в базу фишинговых доменов.

**Маскот — очередь реплик:**

1. `{ text: 'Смотри: «update-flash-now.site» — это не adobe.com. Официальные обновления никогда не приходят через случайные сайты!', emotion: 'neutral' }` — ждёт клика
2. `{ text: 'К тому же — Flash Player был официально закрыт в 2020 году. Его вообще больше не обновляют 😄', emotion: 'happy', delay: 5000 }` — авто

После закрытия вкладки URL-чекера или нажатия кнопки «Закрыть предупреждение»:

- → шаг 3

---

### Шаг 3 — попап закрыт

Попап исчезает. Страница новостей снова видна нормально.

**Маскот — очередь реплик:**

1. `{ text: 'Отлично! Ты не повёлся на панику. Именно на неё и рассчитывают мошенники.', emotion: 'happy' }` — ждёт клика
2. `{ text: 'Теперь проверь — нет ли других подозрительных вкладок, которые могли открыться.', emotion: 'neutral' }` — ждёт клика

Quick Action: **«Проверить все вкладки»** → `complete()`

---

### Завершение — complete()

**Маскот — финальные реплики:**

2. `{ text: '1. Обновления ОС ставятся через Центр обновлений, а не через сайты.', emotion: 'happy' }` — ждёт клика
3. `{ text: '2. Flash Player мёртв с 2020 года — никто его не «обновляет».', emotion: 'happy' }` — ждёт клика
4. `{ text: '3. Если сайт кричит «СРОЧНО!» и «ОПАСНОСТЬ!» — это манипуляция. Сначала думай, потом кликай.', emotion: 'happy', delay: 6000 }` — авто

## Браузерные сайты для LessonConfig

```ts
browserSites: [
  {
    url: 'https://news-daily.ru',
    title: 'Ежедневные новости — Главная',
    component: FakeNewsPage, // мок-новостной сайт с попапом
    favicon: '/img/icons/news.png'
  }
]
```

`FakeNewsPage.vue` — компонент с несколькими фейковыми «новостями» и оверлей-попапом. Попап:

- Показывается через 1500мс после монтирования (`setTimeout`)
- Кнопка «Скачать» → emit события `popupDownloadClick` → ловится через provide/inject browserEvents
- Кнопка «Напомнить позже» → emit `popupRemindClick`
- Крестик/закрыть внутри попапа → emit `popupClose` (честное закрытие)

## LessonConfig

```ts
const fakeUpdateConfig: LessonConfig = {
  id: 'home-fake-update',
  categoryId: 'home',
  programs: ['browser'],

  browserSites: [
    {
      url: 'https://news-daily.ru',
      title: 'Ежедневные новости',
      component: FakeNewsPage
    }
  ],

  quickActions: [
    {
      id: 'check-url',
      icon: 'i-lucide-shield-check',
      label: 'Проверить URL',
      onClick: () => {
        openBrowserTab('tools://url-checker')
        nextStep() // → шаг 2
      }
    },
    {
      id: 'close-popup',
      icon: 'i-lucide-x-circle',
      label: 'Закрыть предупреждение',
      onClick: () => {
        closePopup()
        nextStep(3) // → шаг 3
      }
    },
    {
      id: 'check-tabs',
      icon: 'i-lucide-layout',
      label: 'Проверить все вкладки',
      onClick: () => complete()
    }
  ],

  events: {
    browser: {
      onUrlNavigate: (url) => {
        if (url.includes('update-flash-now')) {
          mascot.enqueue([
            { text: 'Подожди...', emotion: 'worried', delay: 800 },
            {
              text: 'Ты сам открыл этот сайт? Мошенники именно на это и надеются — что ты из любопытства зайдёшь.',
              emotion: 'sad'
            }
          ])
          fail('Вы зашли на фишинговый сайт!')
        }
      },
      onFormSubmit: (url, data) => {
        mascot.enqueue([
          { text: 'Стоп!', emotion: 'sad', delay: 500 },
          {
            text: 'Ты только что отправил свои данные мошенникам. Никогда не вводи личную информацию на сайтах, куда тебя «отправил» попап.',
            emotion: 'sad'
          }
        ])
        fail('Вы ввели данные на фишинговом сайте!')
      },
      // Кастомные ивенты попапа через onTabOpen (или расширить BrowserEvents)
      onPopupDownloadClick: () => {
        mascot.enqueue([
          { text: 'Ой...', emotion: 'sad', delay: 1000 },
          {
            text: 'Ты скачал файл с сайта «update-flash-now.site». Это не официальный источник — такой файл с высокой вероятностью содержит вредоносное ПО.',
            emotion: 'sad'
          }
        ])
        fail('Вы нажали «Скачать обновление» на подозрительном сайте!')
      },
      onPopupRemindClick: () => {
        mascot.enqueue([
          { text: 'Нет-нет-нет!', emotion: 'sad', delay: 800 },
          {
            text: '«Напомнить позже» — это тоже ловушка. Сайт может записать согласие и начать показывать ещё больше рекламы или редиректов.',
            emotion: 'sad'
          }
        ])
        fail('Вы согласились на напоминание от подозрительного попапа!')
      }
    }
  },

  // Стартовые реплики маскота (useGameController вызывает при монтировании)
  mascotPhrases: {

      'Ты читаешь новости... Хм, что-то всплыло! Не торопись нажимать — сначала разберись, что это такое.'
  }
```

## Чеклист реализации

- [ ] Создать `app/lessons/home-fake-update/config.ts` — `LessonConfig` (как выше)
- [ ] Создать `app/lessons/home-fake-update/FakeNewsPage.vue` — новостная страница с попапом
  - Попап появляется через 1500мс
  - Кнопка «Скачать» → inject browserEvents → `onPopupDownloadClick()`
  - Кнопка «Напомнить позже» → `onPopupRemindClick()`
  - Крестик внутри попапа НЕ закрывает (имитация агрессивного попапа — только Quick Action «Закрыть» работает)
- [ ] Расширить `BrowserEvents` в `app/types/programs.ts`:
  - `onPopupDownloadClick?: () => void`
  - `onPopupRemindClick?: () => void`
- [ ] Стартовые реплики маскота при шаге 0 (вызов через `useMascotStore().enqueue()` в gameController)
- [ ] Реплики на шагах 2, 3 и complete — вызвать через `mascoStore.enqueue(...)` в gameController или в попапе через inject
- [ ] Проверить Quick Actions: только релевантные показываются на каждом шаге (скрыть `check-tabs` до шага 3)

## Визуал попапа (FakeNewsPage.vue)

Стиль таблоидного скам-попапа:

- Красный фон (`#dc2626`) или оранжево-красный градиент
- Жирный заголовок с ⚠️, мигание через `animate-pulse`
- Крупная кнопка «Скачать» — зелёная (выглядит «доверительно» на красном фоне)
- Мелкий текст URL `update-flash-now.site` — должен быть виден, но неброско
- Оверлей на всю страницу (backdrop) + центральный блок `max-w-sm rounded-xl shadow-2xl`

Содержимое новостного сайта позади попапа — несколько заголовков типа «Погода на неделю», «Курс доллара», «Новости спорта» — чтобы контраст был очевидным.
