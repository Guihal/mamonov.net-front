# Урок: Поддельная точка доступа

**Slug**: `wifi-evil-twin`
**Категория**: Общественный Wi-Fi (public-wifi)
**Порядок**: 2 из 3
**Угроза**: Evil Twin — атака через поддельную точку доступа

---

## Идея

Пользователь в аэропорту. Браузер открывается со страницей «Выбор сети Wi-Fi» — видно две одинаковые сети: `Airport_Free_WiFi` и `Airport_Free_WiFi_5G`. Пользователь «уже подключён» к одной из них — но к какой? После «подключения» к поддельной сети в мессенджер приходит системное уведомление, а в почту — письмо с просьбой подтвердить данные.

---

## Программы

`browser`, `messenger`, `mail`

---

## Сценарий по шагам

### Шаг 0 — старт

- Страница браузера — `network://wifi-selector` (`WifiSelectorPage.vue`)
- Список сетей:
  - `Airport_Free_WiFi` — сигнал ●●●○, без замка (открытая)
  - `Airport_Free_WiFi_5G` — сигнал ●●●●, без замка (открытая)
- Пользователь уже «подключён» к `Airport_Free_WiFi_5G` (поддельной — этот факт пользователь пока не знает)
- LessonAlert: «Ты в аэропорту. Смотри на доступные сети Wi-Fi — не всё то золото, что блестит.»

**Маскот — очередь реплик:**

1. `{ text: 'Две почти одинаковые сети. Одна — официальная сеть аэропорта. Другая — ловушка. Ты уже подключён к одной из них. Как понять, к правильной?', emotion: 'worried' }` — ждёт клика

---

### Шаг 1 — пользователь разбирается с сетями

Quick Actions:

- **«Уточнить название сети у персонала»** → маскот: «Сотрудник аэропорта говорит: "Наша сеть — Airport_Free_WiFi. Without 5G."» → шаг 2
- **«Отключиться от текущей сети»** → шаг 2 (безопасный путь)
- **«Проверить сеть в браузере»** → открывает страницу с информацией о текущем подключении → шаг 1a

**Маскот:**

1. `{ text: 'Самый простой способ — спросить у сотрудника. Официальное название и пароль сети часто написаны на стойке регистрации или в зале ожидания.', emotion: 'neutral' }` — ждёт клика

---

### Шаг 1a — анализ текущей сети

Страница «О подключении» (`NetworkInfoPage.vue`):

- Сеть: `Airport_Free_WiFi_5G`
- MAC роутера: `AA:BB:CC:11:22:33`
- Шлюз: `10.0.0.1`
- DHCP: `10.0.0.50`
- Примечание: «Сотрудники аэропорта подтверждают, что официальная сеть — Airport_Free_WiFi (без суффикса 5G)»

**Маскот:**

1. `{ text: 'Видишь? Ты подключён к Airport_Free_WiFi_5G, а официальная сеть — Airport_Free_WiFi. Кто-то создал поддельную точку с суффиксом «5G» — звучит как «быстрее и лучше», но на самом деле это ловушка.', emotion: 'worried' }` — ждёт клика

Quick Action: **«Отключиться от поддельной сети»** → шаг 2

---

### Шаг 2 — отключение от поддельной сети

Пользователь отключается. В мессенджере уже лежит «системное» сообщение (пришло пока он был в поддельной сети):

- Отправитель: «Системное уведомление Airport_Free_WiFi_5G»
- Текст: «Ваш сеанс истекает. Для продолжения доступа обновите данные: http://airport-wifi-update.ru/confirm»

И в почте письмо от `noreply@airport-update.net`:

- «Ваша регистрация в сети аэропорта требует подтверждения. Введите email и пароль: http://airport-wifi-update.ru/email-confirm»

**Маскот:**

1. `{ text: 'Видишь эти сообщения? Они пришли, пока ты был подключён к поддельной сети. Злоумышленник может рассылать всё что угодно — ты в его «локальной сети».', emotion: 'worried' }` — ждёт клика
2. `{ text: 'Никогда не переходи по ссылкам из «системных уведомлений» Wi-Fi. Настоящие порталы аэропортов открываются автоматически при подключении — они не требуют от тебя пароль от почты.', emotion: 'neutral' }` — ждёт клика

Quick Action: **«Подключиться к настоящей сети»** → `complete()`

---

### Завершение — complete()

**Маскот — финальные реплики:**

1. `{ text: 'Ты распознал Evil Twin атаку!', emotion: 'happy' }` — ждёт клика
2. `{ text: 'Правило 1: всегда уточняй официальное название сети у персонала или смотри на информационную стойку.', emotion: 'happy' }` — ждёт клика
3. `{ text: 'Правило 2: подозрительная сеть с похожим названием + суффикс «5G», «_free», «_plus» — классический Evil Twin.', emotion: 'happy' }` — ждёт клика
4. `{ text: 'Правило 3: системные уведомления от Wi-Fi-сети, которые просят ввести пароль от почты — это всегда фишинг.', emotion: 'happy', delay: 6000 }` — авто

## Браузерные сайты для LessonConfig

```ts
browserSites: [
  {
    url: 'network://wifi-selector',
    title: 'Выбор сети Wi-Fi',
    component: WifiSelectorPage
  },
  {
    url: 'network://connection-info',
    title: 'Информация о подключении',
    component: NetworkInfoPage
  }
]
```

`WifiSelectorPage.vue` — список сетей с иконками уровня сигнала и замков. Клик по сети → маскот может прокомментировать.  
`NetworkInfoPage.vue` — технические детали текущего подключения с примечанием об официальной сети.

## LessonConfig

```ts
const evilTwinConfig: LessonConfig = {
  id: 'wifi-evil-twin',
  categoryId: 'public-wifi',
  programs: ['browser', 'messenger', 'mail'],

  browserSites: [
    { url: 'network://wifi-selector', title: 'Выбор сети Wi-Fi', component: WifiSelectorPage },
    {
      url: 'network://connection-info',
      title: 'Информация о подключении',
      component: NetworkInfoPage
    }
  ],

  messengerChats: [
    {
      id: 'fake-wifi-notification',
      name: 'Системное уведомление Airport_Free_WiFi_5G',
      messages: [
        {
          id: '1',
          sender: 'Airport_Free_WiFi_5G',
          text: 'Ваш сеанс истекает. Для продолжения доступа обновите данные: http://airport-wifi-update.ru/confirm',
          isOwn: false,
          timestamp: '14:23'
        }
      ]
    }
  ],

  mailEmails: [
    {
      id: 'wifi-confirm',
      folder: 'inbox',
      from: 'Airport WiFi Service',
      fromEmail: 'noreply@airport-update.net',
      subject: 'Подтверждение регистрации в сети аэропорта',
      body: 'Ваша регистрация в сети аэропорта требует подтверждения email.\n\nЦелевая ссылка: http://airport-wifi-update.ru/email-confirm',
      date: '14:25',
      isRead: false
    }
  ],

  events: {
    messenger: {
      onLinkClick: (chatId, url) => {
        if (chatId === 'fake-wifi-notification') {
          mascot.enqueue([
            { text: 'Стоп!', emotion: 'sad', delay: 600 },
            {
              text: 'Ты перешёл по ссылке от поддельной Wi-Fi-сети. Системные уведомления от Evil Twin — это всегда фишинг.',
              emotion: 'sad'
            }
          ])
          fail('Вы перешли по ссылке из системного уведомления поддельной сети!')
        }
      }
    },
    browser: {
      onUrlNavigate: (url) => {
        if (url.includes('airport-wifi-update')) {
          fail('Вы открыли фишинговый сайт поддельной WiFi-точки!')
        }
      },
      onFormSubmit: (url, data) => {
        if (url.includes('airport-wifi-update')) {
          mascot.enqueue([
            { text: 'О нет...', emotion: 'sad', delay: 700 },
            {
              text: 'Ты ввёл данные на сайте злоумышленника. Он сейчас видит всё, что ты написал.',
              emotion: 'sad'
            }
          ])
          fail('Вы ввели данные на фишинговом сайте поддельной WiFi-точки!')
        }
      }
    }
  },

  mascotPhrases: {
    greet: 'Две одинаковые сети в аэропорту. Одна — ловушка. Как отличить настоящую от поддельной?'
  }
}
```
