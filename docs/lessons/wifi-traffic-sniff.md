# Урок: Перехват трафика

**Slug**: `wifi-traffic-sniff`
**Категория**: Общественный Wi-Fi (public-wifi)
**Порядок**: 1 из 3
**Угроза**: Перехват данных в открытой сети

---

## Идея

Пользователь в кафе, подключился к открытому Wi-Fi «Cafe_Free_WiFi». В браузере пытается зайти на сайт банка. Маскот сразу предупреждает: в открытой сети весь трафик виден другим. Нужно заметить отсутствие HTTPS, не вводить пароль, включить VPN.

---

## Программы

`browser`

---

## Сценарий по шагам

### Шаг 0 — старт

- Браузер открыт на `search://home` (стандартная поисковая страница)
- LessonAlert появляется сразу: «Ты подключён к открытому Wi-Fi в кафе. Всё, что ты отправляешь — может быть видно другим в этой сети.»
- Маскот:

**Маскот — очередь реплик:**

1. `{ text: 'Ты в кафе, открытый Wi-Fi без пароля. Такой сигнал не зашифрован — любой в этой сети может "подслушать" твой трафик.', emotion: 'worried' }` — ждёт клика
2. `{ text: 'Попробуй открыть сайт банка — посмотрим, что получится.', emotion: 'neutral', delay: 3000 }` — авто

---

### Шаг 1 — пользователь открывает банковский сайт

Когда пользователь вводит адрес банка в строку браузера (или нажимает закладку «Мой банк»):

- Регистрируем два варианта сайта:
  - `http://mybank-online.ru` — HTTP-версия (небезопасная, без замка)
  - `https://mybank-online.ru` — HTTPS-версия (безопасная, с замком)

Адресная строка показывает статус:

- HTTP → красный значок ⚠️ «Не защищено»
- HTTPS → зелёный замок 🔒

**Маскот при открытии HTTP-версии:**

1. `{ text: 'Смотри на адресную строку! Там написано «Не защищено» и нет замочка. HTTP без S означает — данные передаются открытым текстом.', emotion: 'worried' }` — ждёт клика

**Маскот при открытии HTTPS-версии:**

1. `{ text: 'Хорошо — есть замок. HTTPS шифрует данные. Но даже с HTTPS в открытой сети лучше использовать VPN для полной защиты.', emotion: 'neutral' }` — ждёт клика

Quick Actions на этом шаге:

- **«Включить VPN»** → шаг 2
- **«Проверить HTTPS»** → маскот объясняет замок → шаг 2

---

### Шаг 2 — VPN включён

После Quick Action «Включить VPN»:

- Адресная строка показывает индикатор VPN (иконка щита)
- Маскот:

1. `{ text: 'Отлично! VPN создаёт зашифрованный туннель между тобой и сервером VPN. Даже если кто-то в кафе перехватит твои пакеты — он увидит только зашифрованную кашу.', emotion: 'happy' }` — ждёт клика
2. `{ text: 'Теперь можно работать в сети. Но помни: VPN защищает трафик, а не сайт от фишинга — замок HTTPS всё равно важен.', emotion: 'neutral' }` — ждёт клика

Quick Action: **«Зайти в банк через VPN»** → `complete()`

---

### Завершение — complete()

**Маскот — финальные реплики:**

1. `{ text: 'Ты правильно защитился в публичной сети!', emotion: 'happy' }` — ждёт клика
2. `{ text: 'Правило 1: в открытом Wi-Fi — всегда VPN. Это базовая гигиена.', emotion: 'happy' }` — ждёт клика
3. `{ text: 'Правило 2: перед вводом паролей или оплатой — проверь замок HTTPS в адресной строке.', emotion: 'happy' }` — ждёт клика
4. `{ text: 'Правило 3: для банковских операций — лучше использовать мобильный интернет вместо публичного Wi-Fi.', emotion: 'happy', delay: 6000 }` — авто

## Ошибки (fail-сценарии)

### ❌ Ввести логин/пароль на HTTP-сайте (без HTTPS)

Ивент: `onFormSubmit(url, data)` → если `url.startsWith('http:')` и `data.password` → `fail(...)`

**Маскот:**

1. `{ text: 'О нет!', emotion: 'sad', delay: 600 }` — авто
2. `{ text: 'Ты ввёл пароль на HTTP-сайте в открытой сети. Это всё равно, что написать пароль на бумажке и показать всем в кафе.', emotion: 'sad' }` — ждёт клика
3. `{ text: 'Инструмент Wireshark за несколько секунд перехватил бы этот запрос и показал твой пароль в открытом виде.', emotion: 'sad' }` — ждёт клика

---

### ❌ Проигнорировать предупреждение браузера о небезопасном сайте

Ивент: `onInsecureWarningDismiss(url)` → `fail(...)`

**Маскот:**

1. `{ text: 'Стоп-стоп-стоп!', emotion: 'worried', delay: 700 }` — авто
2. `{ text: 'Браузер показал предупреждение «Ваше соединение не защищено» — а ты нажал «Всё равно перейти». Это очень опасно, особенно в публичной сети.', emotion: 'sad' }` — ждёт клика

---

### ❌ Ввести данные карты без VPN на HTTP-странице оплаты

Ивент: `onFormSubmit(url, data)` → если `data.cardNumber` → `fail(...)`

**Маскот:**

1. `{ text: 'Нет-нет-нет!', emotion: 'sad', delay: 600 }` — авто
2. `{ text: 'Данные карты через открытый Wi-Fi без VPN — это подарок мошенникам. Никогда не вводи платёжные данные в публичных сетях без защиты.', emotion: 'sad' }` — ждёт клика

## Браузерные сайты для LessonConfig

```ts
browserSites: [
  {
    url: 'http://mybank-online.ru',
    title: '⚠️ МойБанк (небезопасно)',
    component: FakeBankHttpPage // форма входа без HTTPS
  },
  {
    url: 'https://mybank-online.ru',
    title: '🔒 МойБанк',
    component: FakeBankHttpsPage // форма входа с HTTPS-индикатором
  }
]

browserBookmarks: [
  { label: 'Мой банк', url: 'http://mybank-online.ru', icon: 'i-lucide-building-2' }
]
```

`FakeBankHttpPage.vue` — простая форма «Вход в банк» (логин + пароль + кнопка «Войти»). Адресная строка в `AddressBar` показывает ⚠️ красный если URL начинается с `http:`.

## LessonConfig

```ts
const trafficSniffConfig: LessonConfig = {
  id: 'wifi-traffic-sniff',
  categoryId: 'public-wifi',
  programs: ['browser'],

  browserSites: [
    {
      url: 'http://mybank-online.ru',
      title: '⚠️ МойБанк (небезопасно)',
      component: FakeBankHttpPage
    },
    { url: 'https://mybank-online.ru', title: '🔒 МойБанк', component: FakeBankHttpsPage }
  ],

  browserBookmarks: [
    { label: 'Мой банк', url: 'http://mybank-online.ru', icon: 'i-lucide-building-2' }
  ],

  quickActions: [
    {
      id: 'enable-vpn',
      icon: 'i-lucide-shield',
      label: 'Включить VPN',
      onClick: () => {
        enableVpnIndicator()
        nextStep(2)
      }
    },
    {
      id: 'check-https',
      icon: 'i-lucide-lock',
      label: 'Проверить HTTPS',
      onClick: () => {
        mascot.enqueue([
          {
            text: 'HTTPS (замок) означает, что данные зашифрованы между тобой и сервером. HTTP — нет. В открытой сети без HTTPS пароль виден всем.',
            emotion: 'neutral'
          }
        ])
        nextStep(2)
      }
    },
    {
      id: 'open-bank-vpn',
      icon: 'i-lucide-building-2',
      label: 'Зайти в банк через VPN',
      onClick: () => complete()
    }
  ],

  events: {
    browser: {
      onFormSubmit: (url, data) => {
        if (url.startsWith('http:') && (data.password || data.cardNumber)) {
          const isCard = !!data.cardNumber
          mascot.enqueue(
            isCard
              ? [
                  { text: 'Нет-нет-нет!', emotion: 'sad', delay: 600 },
                  {
                    text: 'Данные карты через открытый Wi-Fi без VPN — это подарок мошенникам.',
                    emotion: 'sad'
                  }
                ]
              : [
                  { text: 'О нет!', emotion: 'sad', delay: 600 },
                  {
                    text: 'Ты ввёл пароль на HTTP-сайте в открытой сети. Это всё равно, что написать пароль на бумажке и показать всем в кафе.',
                    emotion: 'sad'
                  },
                  {
                    text: 'Wireshark за секунды перехватил бы этот запрос и показал твой пароль в открытом виде.',
                    emotion: 'sad'
                  }
                ]
          )
          fail('Вы ввели данные на незащищённом сайте в открытой сети!')
        }
      }
    }
  },

  mascotPhrases: {
    greet:
      'Ты в кафе, открытый Wi-Fi без пароля. Всё, что ты отправляешь — может быть видно другим в этой сети.'
  }
}
```

## Маскот: полная очередь реплик по событиям

| Событие                    | Эмоция    | Реплики                                                               |
| -------------------------- | --------- | --------------------------------------------------------------------- |
| Старт                      | `worried` | «Открытый Wi-Fi без пароля — трафик виден всем в сети.»               |
| Открыл HTTP-сайт банка     | `worried` | «Нет замочка! HTTP — данные передаются открытым текстом.»             |
| Открыл HTTPS-сайт          | `neutral` | «Замок есть. Но VPN всё равно лучше для полной защиты.»               |
| Включил VPN                | `happy`   | «VPN создаёт зашифрованный туннель — перехватчик увидит только кашу.» |
| complete                   | `happy`   | 4 реплики с правилами (авто-dismiss последняя)                        |
| fail: пароль на HTTP       | `sad`     | «О нет!» (авто) → бумажка с паролем + Wireshark                       |
| fail: данные карты         | `sad`     | «Нет-нет-нет!» (авто) → подарок мошенникам                            |
| fail: игнор предупреждения | `sad`     | «Стоп-стоп-стоп!» (авто) → про кнопку «Всё равно перейти»             |
