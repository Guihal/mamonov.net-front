# Урок: Слабый пароль

**Slug**: `home-weak-password`
**Категория**: Дом (home)
**Порядок**: 1 из 3
**Угроза**: Подбор пароля, утечка данных

---

## Идея

Пользователь заходит в браузер на мок-сайт соцсети `SocialPage.vue`. Поверх страницы появляется баннер: **«Внимание! Ваши данные попали в утечку. Для защиты аккаунта смените пароль прямо сейчас»**. Открывается форма смены пароля. Пользователь должен придумать надёжный пароль — система оценивает его силу в реальном времени.

---

## Программы

`browser`

## Сценарий по шагам

### Шаг 0 — старт

- Браузер открыт на `social://mypage` (`SocialPage.vue` — мок VK-подобного сайта)
- Сверху появляется синий баннер-уведомление: «Обнаружена утечка данных. Рекомендуем сменить пароль немедленно.»
- Кнопка «Сменить пароль» в баннере → открывает форму смены пароля на той же странице

**Маскот — очередь реплик:**

1. `{ text: 'Привет! Ты в соцсети... но что-то не так.', emotion: 'neutral', delay: 1500 }` — авто
2. `{ text: 'Твои данные попали в утечку! Это значит, что кто-то мог получить твой пароль. Нужно его сменить — но сделать это правильно.', emotion: 'worried' }` — ждёт клика

---

### Шаг 1 — форма смены пароля

Форма (`ChangePasswordForm`):

- Поле «Текущий пароль»
- Поле «Новый пароль» с индикатором силы (Слабый / Нормальный / Надёжный)
- Поле «Повторите пароль»
- Кнопка «Сохранить»

Индикатор силы пароля (мок-логика):

- **Слабый** (красный): меньше 8 символов, только цифры, известные словари (`123456`, `qwerty`, `password`, `iloveyou`, дата в формате DDMMYYYY)
- **Нормальный** (жёлтый): 8–11 символов, есть буквы и цифры
- **Надёжный** (зелёный): 12+ символов, буквы + цифры + спецсимволы

Quick Actions:

- **«Использовать менеджер паролей»** → `nextStep()` → автоматически вставляет сгенерированный «надёжный» пароль, форма показывает Надёжный, → шаг 2

Придумать пароль самому пользователь может напрямую — просто печатает в поле формы. Маскот комментирует силу в реальном времени через индикатор.

**Маскот при отображении формы (шаг 1):**

1. `{ text: 'Хороший пароль — не имя, не дата рождения и не «qwerty». Длинный, случайный, с символами.', emotion: 'neutral' }` — ждёт клика

---

### Шаг 2 — пользователь ввёл надёжный пароль

Если пользователь сам ввёл пароль с индикатором «Надёжный» и нажал «Сохранить» → `nextStep(2)` → шаг 2.

**Маскот:**

1. `{ text: 'Отлично! Теперь такой пароль подобрать практически невозможно.', emotion: 'happy' }` — ждёт клика
2. `{ text: 'Кстати, используй разные пароли для разных сайтов. Если один утечёт — остальные будут в безопасности.', emotion: 'neutral' }` — ждёт клика

Quick Action: **«Запомнить в менеджер паролей»** → `complete()`

---

### Завершение — complete()

**Маскот — финальные реплики:**

1. `{ text: 'Молодец! Пароль обновлён.', emotion: 'happy' }` — ждёт клика
2. `{ text: 'Правило 1: длина важнее сложности — 16 случайных символов лучше, чем «P@ssw0rd».', emotion: 'happy' }` — ждёт клика
3. `{ text: 'Правило 2: один сайт — один уникальный пароль.', emotion: 'happy' }` — ждёт клика
4. `{ text: 'Правило 3: используй менеджер паролей — он запомнит всё за тебя.', emotion: 'happy', delay: 5000 }` — авто

## Ошибки (fail-сценарии)

### ❌ Ввести слабый пароль и нажать «Сохранить»

Ивент: `onFormSubmit(url, { newPassword })` → проверяем силу → если слабый → `fail('Вы установили слабый пароль!')`

**Маскот:**

1. `{ text: 'О нет...', emotion: 'sad', delay: 800 }` — авто
2. `{ text: 'Пароль «%password%» подбирается за секунды. Хакеры проверяют миллионы вариантов автоматически.', emotion: 'sad' }` — ждёт клика

---

### ❌ Оставить старый пароль (нажать «Отмена» / закрыть форму без сохранения)

Ивент: `onPasswordChangeDismiss` → `fail('Вы не сменили скомпрометированный пароль!')`

**Маскот:**

1. `{ text: 'Подожди!', emotion: 'worried', delay: 600 }` — авто
2. `{ text: 'Ты закрыл форму, не сменив пароль. Если злоумышленник уже знает твой текущий пароль — аккаунт по-прежнему под угрозой.', emotion: 'sad' }` — ждёт клика

---

### ❌ Использовать пароль `123456`, `qwerty`, имя или дату рождения

(Частный случай слабого пароля — отдельная обработка для педагогики)

**Маскот:**

1. `{ text: 'Серьёзно?', emotion: 'sad', delay: 600 }` — авто
2. `{ text: 'Это один из самых популярных паролей в мире. Хакер подберёт его за долю секунды — он есть в каждом словаре для брутфорса.', emotion: 'sad' }` — ждёт клика

## Браузерные сайты для LessonConfig

```ts
browserSites: [
  {
    url: 'social://mypage',
    title: 'МояСоцСеть — Профиль',
    component: SocialPage,
    favicon: '/img/icons/social.png'
  }
]
```

`SocialPage.vue` — имитация профиля в соцсети:

- Баннер-уведомление об утечке появляется сразу при монтировании
- Кнопка в баннере → показывает `ChangePasswordForm` внутри той же страницы
- `ChangePasswordForm` имеет индикатор силы пароля (computed на основе значения поля)
- При submit → emit `passwordChange({ newPassword })` → inject browserEvents → `onFormSubmit`

## LessonConfig

```ts
const weakPasswordConfig: LessonConfig = {
  id: 'home-weak-password',
  categoryId: 'home',
  programs: ['browser'],

  browserSites: [{ url: 'social://mypage', title: 'МояСоцСеть — Профиль', component: SocialPage }],

  quickActions: [
    {
      id: 'use-password-manager',
      icon: 'i-lucide-key-round',
      label: 'Использовать менеджер паролей',
      onClick: () => {
        fillStrongPassword() // заполняет форму сгенерированным паролем
        nextStep()
      }
    },
    {
      id: 'save-password',
      icon: 'i-lucide-save',
      label: 'Запомнить в менеджере',
      onClick: () => complete()
    }
  ],

  events: {
    browser: {
      onFormSubmit: (url, data) => {
        const { newPassword } = data
        if (isWeakPassword(newPassword)) {
          const isObvious = OBVIOUS_PASSWORDS.includes(newPassword.toLowerCase())
          mascot.enqueue(
            isObvious
              ? [
                  { text: 'Серьёзно?', emotion: 'sad', delay: 600 },
                  {
                    text: 'Это один из самых популярных паролей в мире. Хакер подберёт его за долю секунды.',
                    emotion: 'sad'
                  }
                ]
              : [
                  { text: 'О нет...', emotion: 'sad', delay: 800 },
                  {
                    text: `Пароль «${newPassword}» подбирается за секунды. Хакеры проверяют миллионы вариантов автоматически.`,
                    emotion: 'sad'
                  }
                ]
          )
          fail('Вы установили слабый пароль!')
        } else {
          nextStep(2)
        }
      }
    }
  },

  mascotPhrases: {
    greet: 'Твои данные попали в утечку! Нужно сменить пароль — но сделать это правильно.'
  }
}
```
