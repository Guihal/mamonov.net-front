## Описание

Перенести компонент рабочего стола (workbench) из portfolio-new в mamonov.net-front.

## Архитектура

Файлы рабочего стола не загружаются через API — они инжектируются уроком через `provide/inject`.

```ts
// Страница урока или gameController:
provide('workbenchFiles', [
  { name: 'Браузер', programType: 'browser', path: '/browser/google' },
  { name: 'Почта', programType: 'mail', path: '/mail/inbox' }
] satisfies FsFile[])
```

Если `provide` не вызван — рабочий стол пуст. `hidden: true` фильтруется автоматически.
