# Задача 1.4: Перенос программы Explorer

**Статус**: ✅ Выполнено

## Описание

Перенести программу "Проводник" из portfolio-new. В контексте кибербеза Explorer используется для просмотра файлов — пользователь может наткнуться на подозрительные файлы (exe, bat, вирусные вложения). Урок инжектирует содержимое файловой системы.

**Важно**: breadcrumbs навигация — ТОЛЬКО у Explorer. Остальные программы (Browser, Mail, Messenger) НЕ имеют breadcrumbs.

## Архитектура

```
OS/Window (общий)
└── Content → Explorer/index.vue
    ├── Sidebar           ← дерево папок
    ├── Breadcrumbs       ← навигация по пути
    └── FileGrid/FileList ← содержимое папки
```

### Данные (инжектируются уроком)

```ts
interface ExplorerConfig {
  rootFolder: FolderConfig // корневая папка
  events?: ExplorerEvents
}

interface FolderConfig {
  name: string
  children: (FolderConfig | FileEntryConfig)[]
}

interface FileEntryConfig {
  name: string // «report.xlsx», «virus.exe»
  icon?: string
  isMalicious?: boolean // для gameController
  onOpen?: () => void // кастомное действие при открытии
}

interface ExplorerEvents {
  onFileOpen?: (path: string, file: FileEntryConfig) => void
  onFolderOpen?: (path: string) => void
}
```

### Базовая файловая система (всегда)

```
📁 Рабочий стол
📁 Документы
   📁 Проекты
📁 Загрузки
📁 Изображения
```

Урок расширяет через `lessonConfig.explorerFiles`.

## Стиль (macOS Finder-like)

### Sidebar

```
width: ~200px                 → w-[200px]
background: #f8fafc           → bg-slate-50
border-right: 1px #e2e8f0     → border-r border-slate-200
```

### Breadcrumbs

`UBreadcrumb` из Nuxt UI — подходит напрямую.

### FileGrid

```
background: #ffffff           → bg-white
padding: 12px                 → p-3
gap: 8px                      → gap-2
```

Иконки файлов: `UIcon` (lucide иконки по расширению) + текст `text-xs`.
