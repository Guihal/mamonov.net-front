# Задача 1.5: Настройка цветовой схемы (macOS light)

**Статус**: ✅ Выполнено

## Описание

Настроить цветовую схему в стиле macOS light mode. Убрать pixel-box стиль. Использовать Nuxt UI дизайн-систему без глубокой кастомизации там, где компоненты и так выглядят нормально.

> **Nuxt UI v3 не поддерживает переключение темы per-page/per-component.** `app.config.ts` задаёт глобальные `primary`/`neutral` цвета для всего приложения. OS-компоненты (окна, taskbar, браузер, мессенджер) визуально отличаются от остального сайта — стилизуются через Tailwind-классы и CSS-переменные, не через переопределение Nuxt UI tokens.

### Глобальная тема Nuxt UI (`app.config.ts`)

```ts
export default defineAppConfig({
  ui: {
    colors: {
      primary: 'violet', // акцент сайта (#a57ce9 → violet-400)
      neutral: 'slate' // нейтральная палитра (серые)
    }
  }
})
```
