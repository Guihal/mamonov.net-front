# Задача 1.11: Создание useOsIsolated

**Статус**: ✅ Выполнено

## Описание

Флаг изоляции OS от Nuxt-роутера. Когда OS работает внутри урока (`/app/[category]/[lesson]`), она НЕ должна менять URL при переключении окон — иначе Nuxt попытается навигировать на несуществующий роут и сломает страницу.

Во время уроков `isIsolated = true` всегда. Устанавливается при монтировании страницы урока, сбрасывается при размонтировании.
