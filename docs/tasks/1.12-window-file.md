# Задача 1.12: Система файлов окон (File/FsFile)

**Статус**: ✅ Выполнено

## Описание

Внутри OS роутинг окон работает через `FsFile` объекты. Окна открываются по файлу, каждый файл привязан к `ProgramType`, по которому `Window/Content.vue` определяет какой компонент рендерить.

## Структура FsFile

```ts
// shared/types/Program.d.ts
type ProgramType = 'explorer' | 'browser' | 'mail' | 'messenger'

// shared/types/Entity.d.ts
type Entity = {
  name: string
  programType: ProgramType
  hidden?: boolean
}

// shared/types/FsFile.d.ts
type FsFile = Entity & {
  path: string // внутренний путь, напр. '/browser/google' или '/mail/inbox'
}

// Window.d.ts
type WindowFile = FsFile | null
```

### Связь с программами

| ProgramType | Компонент                         | Описание            |
| ----------- | --------------------------------- | ------------------- |
| `browser`   | `OS/Programs/Browser/index.vue`   | Браузер с вкладками |
| `mail`      | `OS/Programs/Mail/index.vue`      | Почтовый клиент     |
| `messenger` | `OS/Programs/Messenger/index.vue` | Мессенджер          |
| `explorer`  | `OS/Programs/Explorer/index.vue`  | Проводник файлов    |
