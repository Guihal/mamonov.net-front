# Задача 1.16: Создание компонента Elephant (маскот)

**Статус**: ⬜ Не выполнено

## Описание

Визуальный компонент маскота-мамонтёнка. Живёт в правом нижнем углу экрана на странице урока. Глаза следят за курсором, лицо меняется в зависимости от эмоции (`useMascotStore`). Может показывать пузырь с репликой — по клику переключает на следующую.

Работает **только на клиенте**. Данные (эмоция, реплики, видимость) читаются из `useMascotStore` (задача 1.20).

## Базовые размеры

Все пиксельные значения из макета заданы **относительно опорного размера слоника** 430.49 × 288.24 px. В CSS/JS они пересчитываются в проценты через CSS-переменные:

```scss
// В index.vue (или в отдельном SCSS)
--elephant-base-w: 430.49;
--elephant-base-h: 288.24;
--elephant-width: ...; // реальная ширина (задаётся JS при ресайзе)
--elephant-scale: calc(var(--elephant-width) / var(--elephant-base-w));
```

Все позиции элементов (глаза, рот, щёки, брови) задаются через `left` / `top` в процентах от опорного размера:

```
left: calc(86.85 / 430.49 * 100%) = 20.175%
top:  calc(103.78 / 288.24 * 100%) = 36.003%
```

> Размер слоника и его положение на странице (`right: 20px`, `bottom: 60px`) вынесены в объект-конфиг — обработка разных разрешений будет добавлена позже.

## Слежение за курсором — `useElephantMouse.ts`

Composable для вычисления `translate` зрачков на основе позиции мыши.

### Алгоритм

1. **Позиция слоника** — `elephantEl.getBoundingClientRect()`. Обновляется при:
   - `window resize`
   - При первом вызове `start()`

2. **На каждый `mousemove`** — вычислить прогресс X и Y:

   ```ts
   // X
   const elephantLeft = rect.left // из getBoundingClientRect
   const progressX = (((mouseX - elephantLeft) / rect.width) * 100) / 4

   // Y
   const elephantTop = rect.top
   const progressY = (((mouseY - elephantTop) / rect.height) * 100) / 4
   ```

   > Деление на 4 — чтобы зрачок не уходил слишком далеко от центра глаза.

3. **Результат** — `translate(${progressX}%, ${progressY}%)` на зрачке.

4. **Без клемпинга** — `progressX` / `progressY` **не ограничиваются** `Math.min` / `Math.max`. Зрачок может выходить за пределы глаза, чтобы следить за курсором в любом направлении (сверху, снизу, справа).

### API

```ts
export function useElephantMouse(elephantRef: Ref<HTMLElement | null>) {
  const translateX = ref(0)  // в процентах
  const translateY = ref(0)

  function start() { ... }   // addEventListener('mousemove') + addEventListener('resize')
  function stop() { ... }    // removeEventListener обоих

  return { translateX, translateY, start, stop }
}
```

- `elephantRef` — ref на корневой DOM-элемент слоника (для `getBoundingClientRect()`)
- `start()` подписывается на `mousemove` + `resize` (пересчёт rect)
- `stop()` отписывается от обоих

### Где вызывается

`start()` и `stop()` вызываются **внутри компонента `Elephant/index.vue`** (не снаружи):

```ts
// Elephant/index.vue
const elephantRef = ref<HTMLElement | null>(null)
const { translateX, translateY, start, stop } = useElephantMouse(elephantRef)

onMounted(() => start())
onUnmounted(() => stop())
```

Страница урока просто монтирует `<Elephant />` — подписка на `mousemove`/`resize` инкапсулирована внутри компонента.

### Использование в Eye.vue

```vue
<div
  class="pupil"
  :style="{
    transform: `translate(${translateX}%, ${translateY}%)`
  }"
/>
```

> В состоянии `happy` и `worried` зрачки не следят за курсором — translate игнорируется (фиксированные позиции или сердечки).

## Пузырь реплики (SpeechBubble.vue)

Показывается когда `mascotStore.hasText === true`. По клику вызывает `mascotStore.dismissPhrase()`.

Если у текущей реплики есть `delay` — пузырь уходит автоматически (таймер в сторе). Если нет — ждёт клика.

### Стиль

```
bg-white border border-slate-200 rounded-lg shadow-md p-2 text-xs
```

Позиция — над слоником (сверху-правее).

---

## Анимации

Все переходы — `transition: 1s`:

- Смена позиции глаз (left/top)
- Смена размера зрачка
- Появление/исчезновение век (opacity + translateY)
- Появление/исчезновение бровей (opacity)
- Смена рта (opacity crossfade между SVG)
- Смена цвета щёчек (fill transition)

Translate зрачков от курсора — **без transition** (мгновенная реакция на mousemove).
