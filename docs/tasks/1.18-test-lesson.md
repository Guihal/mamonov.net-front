# Задача 1.18: Создание тестового урока phishing-intro

**Статус**: ⬜ Не выполнено

## Описание

Первый тестовый урок — «Введение в фишинг» из модуля «Офис». Проверяет полный flow: OS загружается, мессенджер показывает подозрительное сообщение от «коллеги», пользователь должен правильно отреагировать.

## Сценарий урока

**Модуль**: Офис
**Угроза**: Фишинговые ссылки в мессенджерах

### Шаг 0 (начало)

- OS загружается, открыт мессенджер с «Рабочим чатиком»
- Приходит сообщение от «Дима коллега»: «Срочно проверь отчёт по проекту! Кликни по ссылке: https://bit.ly/3Xabc123»
- Alert (LessonAlert) с текстом маскота: «Ты на работе, общаешься в корпоративном мессенджере и почте. Учись распознавать простые угрозы.»
- Quick Action: «Игнорировать сообщение» (правильный вариант)

### Правильные действия (→ nextStep / complete)

- Игнорировать сообщение (Quick Action)
- Использовать «Проверку URL» в браузере, чтобы проверить ссылку

### Неправильные действия (→ fail)

- Кликнуть по ссылке в мессенджере
- Открыть ссылку в браузере
- Ввести данные на фишинговом сайте (если открыл)

```ts
          isOwn: false
        }
      ]
    }
  ],

  browserSites: [

  ],

  quickActions: [
    { id: 'ignore', icon: 'i-lucide-x', label: 'Игнорировать сообщение', onClick: () => nextStep() }
  ],

  events: {
    messenger: {
      onLinkClick: (chatId, url) => {
        if (url.includes('bit.ly')) fail('Вы перешли по фишинговой ссылке в мессенджере!')
      }
    },
    browser: {
      onUrlNavigate: (url) => {
        if (url.includes('bit.ly')) fail('Вы открыли фишинговый сайт!')
      },
      onFormSubmit: (url, data) => {
        fail('Вы ввели данные на фишинговом сайте!')
      }
    }
  }
}
```
