# Задача 1.1: Перенос компонента Window

**Статус**: ✅ Выполнено

## Описание

Перенести компонент окна из portfolio-new в mamonov.net-front.

**Стиль** — macOS Chrome Desktop (данные из Figma):

### Размеры и позиция окна

```
background: #ffffff
border-radius: 10px          → rounded-[10px]
width: 1048px (начальный)
height: 526px (начальный)
```

### Title bar (общий для всех программ)

```
background: #dfe1e5          → bg-[#dfe1e5]
height: 36px                 → h-9
border-bottom: 1px #dadce0   → border-b border-[#dadce0]
```

> Title bar — общий для всех программ. Содержит только traffic lights (слева) и название программы (по центру). Вкладки (tabs) — внутри content конкретной программы (Browser).

### Traffic lights (кнопки управления слева)

Кастомные `<button>`, **не UButton** — слишком мелкие (11×11px).

| Кнопка   | Fill      | Stroke    | CSS                                             |
| -------- | --------- | --------- | ----------------------------------------------- |
| Close    | `#ee6b5f` | `#e93f30` | `bg-[#ee6b5f] ring-1 ring-inset ring-[#e93f30]` |
| Minimize | `#f5be50` | `#f2ad21` | `bg-[#f5be50] ring-1 ring-inset ring-[#f2ad21]` |
| Expand   | `#62c655` | `#47ac3a` | `bg-[#62c655] ring-1 ring-inset ring-[#47ac3a]` |

```
width: 11px; height: 11px   → size-[11px]
border-radius: 50%           → rounded-full
gap между кнопками: ~8px     → gap-2
```

**Важно**: `breadcrumbs.vue` НЕ переносим в общий header окна. Breadcrumbs — только у Explorer (внутри программы). Общий header остаётся: название программы + traffic light кнопки слева.

**Лоадер НЕ нужен** — файлы описаны в коде как объекты, нет асинхронной загрузки.
