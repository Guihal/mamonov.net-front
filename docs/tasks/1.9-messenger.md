# Задача 1.9: Создание программы Messenger

**Статус**: ⬜ Не выполнено

## Описание

Создать программу "Мессенджер" — визуально похож на Telegram. Слева — список чатов (моковые + основной чат урока), справа — окно переписки. Общие компоненты (ChatList, ChatMessage, InputBar) выносятся в программу, ивенты и данные инжектируются через конфиг урока.

## Архитектура

```
OS/Window (общий)
└── Content → Messenger/index.vue
    ├── ChatList        ← список чатов слева
    └── ChatView        ← переписка справа
        ├── ChatHeader  ← имя + аватар собеседника
        ├── MessageList ← сообщения
        └── InputBar    ← поле ввода + кнопка отправки
```

### Данные (инжектируются уроком)

```ts
interface MessengerConfig {
  chats: ChatConfig[] // список чатов (мок + урок)
  events?: MessengerEvents // ивенты из gameController
}

interface ChatConfig {
  id: string
  name: string // название чата / имя собеседника
  avatar?: string // URL аватара
  messages: MessageConfig[] // начальные сообщения
  isGroupChat?: boolean // групповой чат
}

interface MessageConfig {
  id: string
  sender: string // имя отправителя
  senderAvatar?: string
  text: string
  timestamp?: string
  isOwn?: boolean // от «меня» или нет
}

interface MessengerEvents {
  onMessageSend?: (chatId: string, text: string) => void
  onLinkClick?: (chatId: string, url: string) => void
  onChatOpen?: (chatId: string) => void
}
```

### Базовые моковые чаты (всегда)

| Чат             | Описание                                  |
| --------------- | ----------------------------------------- |
| «Рабочий чатик» | Основной чат урока, сюда приходят задания |
| «Общий»         | Моковый шум — пара сообщений              |
| «HR отдел»      | Моковый — объявления                      |

Урок расширяет через `lessonConfig.messengerChats`.

#### 1.9.1 — `useMessengerState.ts` (`app/composables/os/`)

Единый composable для состояния мессенджера. По аналогии с `useMailState.ts` / `useExplorerState.ts` — принимает `MessengerEvents?`, возвращает реактивное состояние + методы.

```ts
// app/composables/os/useMessengerState.ts

      }
    ]
  },

    avatar: undefined,
    isGroupChat: false,
    messages: [
      {
        id: 'hr-1',
        sender: 'HR отдел',
        text: 'Напоминаем: завтра последний день подачи заявления на отпуск на май.',
        timestamp: '2026-04-03T09:00:00',
        isOwn: false
      }
    ]

export function useMessengerState(events?: MessengerEvents) {
  const chats = ref<ChatConfig[]>(deepClone(BASE_CHATS))

  /** ID активного чата (null = ничего не выбрано) */
  const activeChatId = ref<string | null>(null)

  /** Текущий активный чат */
  const activeChat = computed<ChatConfig | null>(() =>
    activeChatId.value ? (chats.value.find((c) => c.id === activeChatId.value) ?? null) : null
  )

  /** Выбрать чат */
  const selectChat = (chatId: string) => {
    activeChatId.value = chatId
    events?.onChatOpen?.(chatId)
  }

  /** Отправить своё сообщение */
  const sendMessage = (text: string) => {
    if (!activeChatId.value || !text.trim()) return
    const chat = chats.value.find((c) => c.id === activeChatId.value)
    if (!chat) return
    const msg: MessageConfig = {
      id: `own-${Date.now()}`,
      sender: 'Я',
      text: text.trim(),
      timestamp: new Date().toISOString(),
      isOwn: true
    }
    chat.messages.push(msg)
    events?.onMessageSend?.(activeChatId.value, text.trim())
  }

  /** Урок динамически добавляет сообщение в чат (например, NPC пишет ответ) */
  const pushMessage = (chatId: string, message: MessageConfig) => {
    const chat = chats.value.find((c) => c.id === chatId)
    if (!chat) {
      console.warn('[useMessengerState] pushMessage: чат не найден', chatId)
      return
    }
    chat.messages.push(deepClone(message))
  }

  /** Урок добавляет свои чаты */
  const addChats = (newChats: ChatConfig[]) => {
    for (const chat of newChats) {
      if (!chats.value.find((c) => c.id === chat.id)) {
        chats.value.push(deepClone(chat))
      }
    }
  }

  /** Обработка клика по ссылке в сообщении */
  const handleLinkClick = (url: string) => {
    if (!activeChatId.value) return
    events?.onLinkClick?.(activeChatId.value, url)
  }

  /** Последнее сообщение чата (для превью в ChatList) */
  const getLastMessage = (chatId: string): MessageConfig | null => {
    const chat = chats.value.find((c) => c.id === chatId)
    if (!chat || chat.messages.length === 0) return null
    return chat.messages[chat.messages.length - 1]
  }

  return {
    chats: readonly(chats),
    activeChatId: readonly(activeChatId),
    activeChat,
    selectChat,
    sendMessage,
    pushMessage,
    addChats,
    handleLinkClick,
    getLastMessage
  }
}

function deepClone<T>(obj: T): T {
  return JSON.parse(JSON.stringify(obj))
}
```

API composable:

| Метод / Свойство  | Тип                                         | Описание                                   |
| ----------------- | ------------------------------------------- | ------------------------------------------ |
| `chats`           | `Readonly<Ref<ChatConfig[]>>`               | Все чаты (базовые + урок)                  |
| `activeChatId`    | `Readonly<Ref<string \| null>>`             | ID выбранного чата                         |
| `activeChat`      | `ComputedRef<ChatConfig \| null>`           | Текущий чат                                |
| `selectChat`      | `(chatId: string) => void`                  | Выбрать чат → `onChatOpen`                 |
| `sendMessage`     | `(text: string) => void`                    | Отправить своё сообщение → `onMessageSend` |
| `pushMessage`     | `(chatId, message) => void`                 | Урок/NPC динамически добавляет сообщение   |
| `addChats`        | `(chats: ChatConfig[]) => void`             | Урок добавляет чаты из `lessonConfig`      |
| `handleLinkClick` | `(url: string) => void`                     | Клик по ссылке → `onLinkClick`             |
| `getLastMessage`  | `(chatId: string) => MessageConfig \| null` | Последнее сообщение (для превью)           |

#### 1.9.5 — `index.vue` (корневой)

По аналогии с Mail — получает конфиг через `inject('lessonConfig')`, создаёт `useMessengerState()`, provide для дочерних.

```vue
<script setup lang="ts">
import type { LessonConfig } from '~/types/lesson'
import { useMessengerState } from '~/composables/os/useMessengerState'

const lessonConfig = inject<LessonConfig | null>('lessonConfig', null)
const events = lessonConfig?.events?.messenger ?? {}

const messengerState = useMessengerState(events)

// Регистрация чатов из урока
if (lessonConfig?.messengerChats) messengerState.addChats(lessonConfig.messengerChats)

// provide для дочерних компонентов
provide('messengerState', messengerState)
</script>

<template>
  <div class="messenger">
    <OSProgramsMessengerChatList />
    <OSProgramsMessengerChatView />
  </div>
</template>

<style lang="scss">
.messenger {
  display: flex;

  height: 100%;
  overflow: hidden;
}
</style>
```

#### 1.9.10 — `ChatMessage.vue` (пузырь)

Одно сообщение. Два режима: входящее (аватар + пузырь слева, `bg-white`) и исходящее (пузырь справа, `bg-violet-500`).

```vue
<script setup lang="ts">
import type { MessageConfig } from '~/types/programs'

const props = defineProps<{ message: MessageConfig }>()

const messengerState = inject('messengerState') as ReturnType<typeof useMessengerState>

const formattedTime = computed(() => {
  if (!props.message.timestamp) return ''
  const d = new Date(props.message.timestamp)
  return `${String(d.getHours()).padStart(2, '0')}:${String(d.getMinutes()).padStart(2, '0')}`
})

const handleClick = (e: MouseEvent) => {
  const target = e.target as HTMLElement
  if (target.tagName === 'A' && target.getAttribute('href')) {
    e.preventDefault()
    messengerState.handleLinkClick(target.getAttribute('href')!)
  }
}
</script>

<template>
  <div class="chat-message" :class="{ 'chat-message--own': message.isOwn }" @click="handleClick">
    <!-- Аватар входящего -->
    <UAvatar
      v-if="!message.isOwn"
      :alt="message.sender"
      :src="message.senderAvatar"
      size="sm"
      class="chat-message__avatar"
    />

    <div class="chat-message__bubble">
      <!-- Имя отправителя (только во входящих) -->
      <span v-if="!message.isOwn" class="chat-message__sender">
        {{ message.sender }}
      </span>
      <span class="chat-message__text">{{ message.text }}</span>
      <span class="chat-message__time">{{ formattedTime }}</span>
    </div>
  </div>
</template>
```

**Стиль:**

- Контейнер входящего: `flex gap-2 items-end max-w-[75%]`
- Контейнер исходящего: `flex gap-2 items-end max-w-[75%] ml-auto flex-row-reverse`
- Пузырь входящего: `bg-white border border-slate-200 rounded-lg p-2`
- Пузырь исходящего: `bg-violet-500 text-white rounded-lg p-2`
- Имя: `text-sm font-medium text-black block mb-0.5`
- Текст: `text-xs leading-4 text-[#314158]` (входящие) / `text-xs leading-4 text-white` (исходящие)
- Время: `text-[10px] text-slate-400 mt-0.5 block text-right` / `text-[10px] text-white/60` (исходящие)
- Аватар: `flex-shrink-0`

#### 1.9.11 — `InputBar.vue`

Поле ввода + кнопка отправки. Enter = отправка.

```vue
<script setup lang="ts">
const messengerState = inject('messengerState') as ReturnType<typeof useMessengerState>

const inputText = ref('')

const send = () => {
  if (!inputText.value.trim()) return
  messengerState.sendMessage(inputText.value)
  inputText.value = ''
}
</script>

<template>

    />
    <UButton
      icon="i-lucide-send"
```
