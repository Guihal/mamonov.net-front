# Задача 1.2: Перенос компонента Taskbar

**Статус**: ✅ Выполнено

## Описание

Перенести компонент нижней панели (taskbar) из portfolio-new в mamonov.net-front.

## Стиль из Figma

### Taskbar контейнер

```
background: linear-gradient(#f8f4f466, #f8f4f466)  → bg-[#f8f4f466]
border-radius: 20px                                  → rounded-[20px]
width: 1382px (почти на всю ширину, margin 29px)
height: 82px                                          → h-[82px]
position: absolute; bottom: 3px; left: 29px
```

> Кастомный компонент. Nuxt UI не имеет подходящего готового компонента для dock/taskbar.

### Разделитель

```
border: 1px solid #0d0b0b     → border-l border-[#0d0b0b]
height: 59px                    → h-[59px]
vertical
```

### Аватар пользователя (кнопка Пуск)

```
width: 48px; height: 48px      → size-12
border-radius: 100px            → rounded-full
background: #c09aff             → bg-[#c09aff]
icon: user-round (28×28px)      → <UIcon name="i-lucide-user-round" size="28" />
```

### Иконки Dock

Размеры: `42–65px` ширина, `49–57px` высота. Средний размер ~`50px`.

| Иконка     | Размер    |
| ---------- | --------- |
| Finder     | `42×53px` |
| Chrome     | `45×53px` |
| Note       | `49×54px` |
| PowerPoint | `47×51px` |
| Publisher  | `49×50px` |
| Word       | `52×49px` |
| Access     | `49×53px` |
| WordPress  | `43×53px` |
| Documents  | `58×56px` |

> Иконки — статические изображения (PNG/SVG). Не UButton.

### Новые требования (отличаются от portfolio-new)

1. **Меню «Пуск»** — кнопка в таскбаре, открывающая Tooltip с меню:
   - Кнопка **«Выйти»** — ведёт на `/app/[categorySlug]/` (на уровень выше из урока к списку уроков категории)
   - В будущем меню будет расширяться
2. **Закреплённые программы** — программы, используемые в текущем уроке, должны быть **закреплены** в таскбаре (видны всегда, даже когда закрыты). Это новая функциональность, в portfolio-new не реализована — окна показывались только пока открыты.
3. **Tooltip** — использовать компонент Tooltip из portfolio-new для всплывающего меню пуска и превью окон.
