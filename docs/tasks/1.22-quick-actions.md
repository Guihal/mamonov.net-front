# Задача 1.22: Панель общих действий (Quick Actions)

**Статус**: ⬜ Не выполнено

## Описание

Создать панель общих действий, которая отображается **вне OS** — вертикальный текст "ДЕЙСТВИЯ" (каждая буква на отдельной строке), прижатый к **левой** стороне экрана. При наведении раскрывается полная менюшка.

## Визуал

Текст "ДЕЙСТВИЯ" разбит на отдельные `<span>`/`<div>`, каждая буква — новая строка:

```
Д
Е
Й
С
Т
В
И
Я
```

- Прижата к **левой** стороне экрана
- `border-radius` только на правый верх и правый низ (`rounded-r-lg`)
- При наведении — выезжает полная менюшка с действиями (справа от текста)
- Каждое действие: иконка + подпись
- При клике — вызывается коллбек из `LessonConfig`
- Коллбек описывается в `gameController.events`

### Стиль из Figma (Quick Action / доп. действие)

```
background: rgba(241,245,249,0.5)  → bg-slate-100/50
border-radius: 8px                  → rounded-lg
icon: 20×20px                       → UIcon size="20"
font: DM Sans 500 14px/20px         → text-sm font-medium leading-5
color: #0f172b                       → text-slate-900
```

> Кастомный компонент. `UButton` может использоваться для отдельных action items с `variant="ghost"` и `color="neutral"`.

### `app/components/OS/QuickActions/index.vue`

Корневой компонент:

- Читает `quickActions` из `inject('lessonConfig')` — данные кладёт туда страница урока через `provide`.
- Если список пуст или `undefined` — не рендерится ничего.
- Вертикальная плашка «ДЕЙСТВИЯ»: каждая буква в отдельном `<span>`.
- `border-radius` только справа: `border-radius: 0 8px 8px 0`.
- При наведении на плашку — показывается `Menu` с флагом `isOpen`.
- `Menu` выезжает через CSS `Transition` (`slide-x`).

```vue
<script setup lang="ts">
import type { LessonConfig } from '~/types/lesson'

// Урок передаёт конфиг через provide('lessonConfig', config) в useGameController
const config = inject<LessonConfig>('lessonConfig')

const actions = computed(() => config?.quickActions ?? [])

const isOpen = ref(false)

// Небольшая задержка при скрытии, чтобы мышь успела перейти на меню
let hideTimer: ReturnType<typeof setTimeout> | null = null

const showMenu = () => {
  if (hideTimer) {
    clearTimeout(hideTimer)
    hideTimer = null
  }
  isOpen.value = true
}

const scheduleHide = () => {
  hideTimer = setTimeout(() => {
    isOpen.value = false
  }, 120)
}
</script>

<template>
  <div
    v-if="actions.length"
    class="quick-actions"
    @mouseenter="showMenu"
    @mouseleave="scheduleHide"
  >
    <!-- Вертикальная плашка -->
    <div class="quick-actions__trigger">
      <span v-for="(letter, i) in 'ДЕЙСТВИЯ'.split('')" :key="i" class="quick-actions__letter">{{
        letter
      }}</span>
    </div>

    <!-- Выезжающее меню -->
    <Transition name="qa-slide">
      <div
        v-if="isOpen"
        class="quick-actions__menu-wrap"
        @mouseenter="showMenu"
        @mouseleave="scheduleHide"
      >
        <OSQuickActionsMenu :actions="actions" />
      </div>
    </Transition>
  </div>
</template>

<style lang="scss">
.quick-actions {
  position: fixed;
  left: 0;
  top: 50%;
  transform: translateY(-50%);
  z-index: 200; /* выше OS-окон (z-index окна ~100), ниже модалей */
  display: flex;
  flex-direction: row;
  align-items: center;

  /* Плашка с буквами */
  &__trigger {
    display: flex;
    flex-direction: column;
    align-items: center;
    padding: 12px 8px;
    background: rgba(241, 245, 249, 0.85);
    backdrop-filter: blur(8px);
    border-radius: 0 8px 8px 0;
    box-shadow: 2px 0 12px rgba(0, 0, 0, 0.08);
    cursor: default;
    user-select: none;
    gap: 3px;
  }

  &__letter {
    font-family: 'DM Sans', sans-serif;
    font-size: 11px;
    font-weight: 600;
    line-height: 14px;
    color: #0f172b;
    display: block;
    text-align: center;
  }

  &__menu-wrap {
    /* Меню появляется сразу справа от плашки */
    position: absolute;
    left: 100%;
    top: 50%;
    transform: translateY(-50%);
  }
}

/* Анимация выезда меню слева */
.qa-slide-enter-active,
.qa-slide-leave-active {
  transition:
    opacity 0.18s ease,
    transform 0.18s ease;
}

.qa-slide-enter-from,
.qa-slide-leave-to {
  opacity: 0;
  transform: translateY(-50%) translateX(-8px);
}
</style>
```

### Интеграция в страницу урока

В `pages/app/[category]/[lesson]/index.vue` компонент размещается **вне** OS-зоны, прямо в template рядом с `<OSTaskbar />`:

```vue
<template>
  <div class="lesson-page">
    <!-- OS-зона -->
    <OSWorkbench />
    <OSWindowView />
    <OSTaskbar />
    <OSTaskbarTooltips />

    <!-- QuickActions — вне OS, поверх всего -->
    <OSQuickActions />
  </div>
</template>
```

`provide('lessonConfig', config)` уже вызывается внутри `useGameController(config)` (задача 1.13). Поэтому `OSQuickActions` получает данные автоматически через `inject`.

#### Пример конфига урока с QuickActions

```ts
// pages/app/[category]/[lesson]/index.vue
const config: LessonConfig = {
  id: 'phishing-intro',
  categoryId: 'office',
  programs: ['browser', 'mail'],
  quickActions: [
    {
      id: 'call-colleague',
      icon: 'i-lucide-phone',
      label: 'Позвонить коллеге',
      onClick: () => {
        // открыть окно мессенджера или показать подсказку
        useCreateAndRegisterWindow('/messenger/colleague')
      }
    },
    {
      id: 'open-docs',
      icon: 'i-lucide-file-text',
      label: 'Открыть документацию',
      onClick: () => {
        useCreateAndRegisterWindow('/browser/docs')
      }
    }
  ],
  events: {
    mail: {
      /* ... */
    }
  }
}

useGameController(config)
```
