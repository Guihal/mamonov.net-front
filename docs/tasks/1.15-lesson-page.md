# Задача 1.15: Создание страницы урока

**Статус**: ⬜ Не выполнено

## Описание

Создать страницу урока, которая монтирует OS в изолированном режиме.

## Стиль из Figma

### Фон страницы (OS-зона)

```
background: #f1f1f1            → bg-[#f1f1f1]
border-radius: 20px            → rounded-[20px]  (если фоновое изображение)
размер: 1440×847px (full viewport)
```

Фоновый скриншот (image 47) заполняет всю зону: `object-fit: fill`, `rounded-[20px]`.

### Верхняя полоса (Mac Menu Bar — декоративная)

```
height: 33px                    → h-[33px]
border-radius: 4px              → rounded
background: linear-gradient(#c4c4c466, #c4c4c466)  → bg-[#c4c4c466]
```

> Декоративный элемент macOS menu bar. Иконки: Apple logo, часы, Wi-Fi, батарея, звук — кастомные изображения `24×24px`.

### Alert — информационная подсказка урока (компонент `LessonAlert`)

**Фрейм A — описание задачи:**

```
background: #ffffff             → bg-white
border: 1px solid #e2e8f0       → border border-slate-200
border-radius: 8px              → rounded-lg
width: 866px                    → max-w-[866px] w-full
height: auto (min 76px)
position: centered, над taskbar → mx-auto, bottom: ~64px
```

> Можно реализовать через `UAlert` из Nuxt UI с кастомным slot'ом для аватара. Или полностью кастомный компонент.

**Аватар маскота внутри Alert:**

```
width: 44px; height: 44px       → size-11
border-radius: 999px            → rounded-full
overlay: rgba(0,0,0,0.2)        → кастомный overlay
```

**Текст Alert:**

```
font: DM Sans 500 14px/20px     → text-sm font-medium leading-5
color: #0f172b                   → text-slate-900
```

**Кнопка закрытия:**

```
width: 24px; height: 24px       → UButton size="xs" variant="ghost" icon="i-lucide-x"
border-radius: 6px              → rounded-md (дефолт UButton)
```

### Quick Action (Фрейм B — доп. действие)
