# Задача 1.17: Создание Comic компонента

**Статус**: ⬜ Не выполнено

## Описание

Создать компонент слайд-шоу, который показывается **один раз** — после первого захода пользователя в аккаунт (после регистрации/логина). Рассказывает пользователю о том, что такое кибербезопасность и зачем проходить уроки.

## Поведение

Comic показывается только при выполнении **всех** условий:

- Пользователь только что зарегистрировался/вошёл первый раз
- Куки `is_first` отсутствует или `false`
- Прогресс пользователя равен нулю (нет пройденных уроков)

**После прохождения Comic** → редирект на `/app` (список категорий/модулей).

**Comic один**, захардкочен в компоненте. НЕ принимает пропсы.

### Контент комикса (примерные слайды)

1. Мамонтёнок знакомится с пользователем
2. «В интернете много угроз — фишинг, вирусы, соц. инженерия»
3. «Ты будешь проходить модули — офис, дом, публичная сеть»
4. «Каждая ошибка снимает HP. Будь внимателен!»
5. «Готов? Поехали!»

### Стиль

Полноэкранный белый фон, слайд по центру:

```
bg-white min-h-screen flex items-center justify-center
```

Слайд:

```
max-w-2xl mx-auto
```

Картинка слайда: `rounded-xl shadow-lg` (полная ширина контейнера).

Навигация:

- `UButton variant="ghost"` — «Назад»
- `UButton color="primary"` — «Далее» / «Начать!» (последний слайд)
- Индикатор прогресса: ряд точек (`rounded-full size-2`)
