# Задача 1.24: Страница категорий и уроков

**Статус**: ⬜ Не выполнено

## Описание

Страницы для навигации по категориям и урокам. Читают данные из `useMockDbStore`.

---

## Страницы

### 1. `/app` — Список категорий

Карточки модулей: Офис, Дом, Общественный Wi-Fi. Каждая показывает название, иконку, прогресс (количество пройденных уроков / всего), difficulty. Клик → `/app/categories/:slug`.

### 2. `/app/categories/:categorySlug` — Уроки категории

Список уроков категории. Каждый урок:

- **Разблокирован** — клик → `/app/categories/:categorySlug/:lessonSlug`
- **Заблокирован** — серый, иконка замка, не кликабельный
- **Пройден** — галочка, зелёный

Показывает HP текущей категории (бар сверху или рядом с заголовком).

---

## Логика разблокировки

Урок `n` доступен, если урок `n-1` пройден (`isCompleted`). Первый урок всегда доступен.

Данные из `mockDbStore.isLessonUnlocked(categoryId, slug)`.
