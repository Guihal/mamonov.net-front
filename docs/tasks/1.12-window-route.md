# Задача 1.12: Создание useWindowRoute

**Статус**: ✅ Выполнено

## Описание

Composable для синхронизации внутреннего пути окна. В изолированном режиме (урок) — НЕ трогает Nuxt router. Вне изоляции (если когда-то понадобится) — синхронизирует через `queuedPush`.

> В контексте уроков OS всегда работает в изолированном режиме (`isIsolated = true`), поэтому `router.push` вызываться не будет. Composable нужен для обратной совместимости и на случай если OS будет использована вне уроков.
