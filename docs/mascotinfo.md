Заключение:
у слона есть общее время анимации - 1 секунда, она на всех transition используется.
глаза сделаны просто кругом с бордер F8FAFC

все переходы анимированы (ну тут фиксированные размеры есть, вроде, их можно считать в css просто относительно размера слоника, которые просто в конкретные размеры пересчитываются, т.е. размеры - css переменные).

глаза у него меняют свое положение в зависимости от положения мыши относительно слоненка (типо следят за курсором). Основное положение задается через left/top относительно глаза, а translate мы считаем (напрямую пишем, не переменная) от курсора (типо берем левую сторону, смотрим положение слоника через right 20px (это его фиксированное положение на странице) + var(--elephant-width) (размер тогда хардкодить через js по разрешению), bottom 60px (положение и размер вынесено в объект, который потом самостоятельно я обработаю в зависимости от размера экрана))

translate считается так:

x:
положение слоника: window.innerWidth - right слоника - ширина слоника
дальше просто берем и считаем прогресс Math.max(0, Math.min(25, x курсора / положение слоника \* 100 / 4 )); Делим на 4, чтобы зрачок не выходил за пределы глаза.

и translate X: `${прогресс} %`;

y: то же самое, только с вертикальной позицией.
